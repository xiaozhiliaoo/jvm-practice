# JVM-PRACTICE

整理的JVM的一些资料，包括周志明《深入理解JVM》，葛一鸣《实战JVM》代码以及自己测试Demo。

以及中国所有出版的JVM的书籍整理。还有国外的PPT，论文的汇总。

# 国内出版的JVM的书汇总（总共27本）

4. 《深入理解JVM ＆ G1 GC》 周明耀
5. 《Java虚拟机精讲》 高翔龙

7. 《自己动手写Python虚拟机》
